## What is PGV Mesh?

PGV Mesh is a volunteer-run community network using MeshCore for grid-independent text communication.

## Getting Started with MeshCore

1. **Hardware**: Get compatible hardware (e.g. Heltec V3, T-Echo, etc.).
2. **Flash**: Flash your device via [flasher.meshcore.io](https://flasher.meshcore.io).
   - **Companion**: Flashing for companions (connecting to phone/computer).
   - **Repeater**: Flashing for standalone repeaters/routers.
3. **Pair**: Connect your device via Bluetooth or Wi-Fi to pair with the app.

## Repeater Advanced Tuning (Admin commands)

### Transmission Delays

- **0–1 Neighbors (Sparse/Pockets)**:
  - `set txdelay 0.3`
  - `set direct.txdelay 0.1`
- **2–4 Neighbors**:
  - `set txdelay 0.5`
  - `set direct.txdelay 0.3`
- **5–9 Neighbors**:
  - `set txdelay 1.0`
  - `set direct.txdelay 0.5`
- **10–14 Neighbors (Dense)**:
  - `set txdelay 1.5`
  - `set direct.txdelay 1.0`

### Stability Commands

```
set agc.reset.interval 4
set multi.acks 1
```
